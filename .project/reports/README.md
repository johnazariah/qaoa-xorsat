# Daily Reports

This directory contains daily audit reports generated by the `start-work` skill.

Each file is named `YYYY-MM-DD.md` and contains:
- Executive summary of project health
- Daily changes (commits, features, tests)
- System snapshot (line counts, test health, plan progress)
- Next steps from the project plan

Reports are written for **yesterday** at the start of each work session,
so they capture the full day's work.
